VapingCation 最终用户许可协议

服务条款与用户协议

协议声明

本《最终用户许可协议》（以下简称"本协议"）由 VapingCation 管理团队（以下简称"运营方"）与用户共同缔结。用户通过访问或使用本 Minecraft 服务器即视为已阅读并同意接受本协议全部条款的约束。

* 本协议条款可能不定期更新，更新后的协议自发布之日起生效。用户继续使用服务视为接受修订后的协议。

用户义务

1. 用户应严格遵守 Mojang Studios® 及 Microsoft® 相关用户协议，Minecraft 官方使用指南请参见：https://www.minecraft.net/zh-hans/usage-guidelines#brand

2. 禁止在本服务平台（包括但不限于游戏服务器、腾讯 QQ 群组等）从事以下行为：

  • 商业交易、广告推广及账号交易。
  • 传播淫秽色情、暴力恐怖等违法违规信息。
  • 发布损害国家利益、民族情感的言论。
  • 对游戏服务器或腾讯 QQ 群组的恶意破坏（包括但不限于: 对游戏服务器进行攻击，在腾讯 QQ 群组中刷屏）。

如有发现，欢迎举报（有偿）。

3. 不滥用不属于 Minecraft 原版游戏的 Bug。如有发现，请上报给管理层（有偿）。

4. 若第 2 条是建立在滥用举报或者由举报者被动导致，则举报者此时亦会受到 1~3 天禁言的处罚。

5. 请不要使用多个账户加入腾讯 QQ 群组。

6. 在 VapingCation 合法版中，请不要做出任何影响游戏平衡的行为。

信息安全

1. 用户应妥善保管账户信息，因用户自身原因导致的账户安全问题，运营方不承担任何责任。

同样，运营方不会以任何理由、任何方式公开任何用户的账户信息。

2. 若持有服务器直连 IP 地址，请不要泄露服务器直连 IP 地址（公开测试 IP 除外），若因用户泄露导致服务器遭受攻击，运营方保留追究法律责任的权利。

3. 若成功加入管理层，请不要公开服务器内部信息，无论有意或无意。

  • 发现后将会立即从服务器中移除用户数据。
  • 被移除的用户可以申请再次加入服务器，但不可再次申请加入管理层。
  • 被移除的用户若使用副账户申请加入管理层，发现后会同时对使用的账户禁言 5~10 天。

4. 申请加入管理层被拒绝或者因为特殊原因无法通过，无法再次申请。

5. 虽然管理层会适当管控游戏服务器或腾讯 QQ 群组，但被骗导致的损失我们也无能为力。

6. 维护团队测试时，管理层有权对不慎进入游戏服务器的玩家进行临时性封禁。

  • 此封禁不是永久的，在测试完成后会清除在此次测试中的所有封禁记录。

    • 违规处理

    1. 违反本协议条款的用户，运营方有权根据情节采取以下措施:

    • a. 临时或永久封禁游戏账户。
    • b. 在腾讯 QQ 群组中移除。
    • c. 情节严重的，将会追究法律责任。

    2. 对服务器进行攻击或破坏的，运营方将永久列入黑名单并保留司法追诉权利。
